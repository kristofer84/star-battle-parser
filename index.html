Parameters Reset defaults
Color distance Max RGB distance to merge cells into same region. ↑ merges similar colors.
Star threshold Min % of scan area that must be dark to call it a star. ↓ detects more stars.
X mark threshold Min % of scan area that must be red to call it an X. ↓ detects more X marks.
Grid sensitivity Min % of peak density to count a row as grid. ↓ is more permissive.
Mark scan size % of cell half-width scanned for marks. ↑ scans a larger area.
